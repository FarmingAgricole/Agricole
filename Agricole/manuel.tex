\section{Manuel Utilisateur}
\label{sec:manuel}
\subsection{Introduction}Cette section à pour but d'expliquer les fonctionnalitées du programme.

\subsection{Démarrage du jeu}
\subsubsection{Choix des paramètres du jeu}Après lancement du programme, le jeu commence.
Il est alors demandé à l'utilisateur le nom qu'il souhaite donner à sa ferme virtuelle et la difficultée qu'il désire.
\subsubsection{}

\subsection{Déroulement du jeu}
\subsubsection{Début du jeu}Après le choix des paramètres, l'utilisateur arrive alors sur la fenêtre principale du jeu.
Différents éléments lui sont affichés:
\vspace{3mm}\begin{itemize}
\item la ferme
\item le bouton pour aller au magasin
\item le bouton pour gérer les animaux
\item le bouton pour gérer les cultures.
\end{itemize}

\subsubsection{La Ferme}L'utilisateur peut faire état de la ferme, différentes informations lui sont affichés pour indiquer l'avancement de celle-ci:

Il lui appartient de cliquer les cases représentant l'élément choisi pour que le programme y affiche les informations relatées.
Voici les différents éléments :
\vspace{3mm}\begin{itemize}
\item les 4 différents enclos contiennent chacun 4 différentes cases (4x4 donc), si sélectionnée une case affiche l'animal, son nom, ses points de vie, son état
\item les parcelles cultivables, si séléctionnées, seront affichés le type de culture, ses points de vie, son état, si elle a été arrosée ou non
\item le garage, si sélectionné, les différents véhicules disponibles dans la fermes seront affichés(voir achat de véhicule dans la section shop)
\end{itemize}

Des boutons pour permettre à l'utilisateur d'effectuer certaines actions lui sont également disponibles:
\vspace{3mm}\begin{itemize}
\item le bouton "<Passer la journée">, permet de passer à la journée suivante
\end{itemize}

D'autres informations affichées sont non cliquables et servent uniquement à informer l'utilisateur:
\vspace{3mm}\begin{itemize}
\item le montant de l'argent que possède le fermier
\item le nombre de jours passés depuis le début du jeu
\end{itemize}

\subsubsection{Le Magasin}
Après avoir cliqué sur le bouton Shop, l'utilisateur est dirigé vers le magasin.
	\paragraph{Achats} 
		\subparagraph{} Dans cette fenêtre il a la possibilité de dépensé l'argent du fermier pour finaliser divers achat en cliquant sur l'élément choisit, ces différents éléments sont:
			\vspace{3mm}\begin{itemize}
				\item les 4 différents animaux: la poule, la vache, le mouton et le cochon, chacun ont leur propre prix et manière d'être élevé
				\item les 2 différentes cultures: la fraise et la carotte
				\item les véhicules et leurs niveaux
				\item les ressources nécessaires à la ferme: le foin et les graines
			\end{itemize}

Pour plus d'information sur les caractéristiques des différents éléments d'une même catégories veuillez consulter le Scénario de jeu.

	\paragraph{Ventes}
		\subparagraph{} L'utilisateur peut également vendre ses ressources via ce menu, les différentes ventes possible et leurs boutton sont:
			\vspace{3mm}\begin{itemize}
				\item le lait
				\item les oeufs
				\item la laine
				\item la viande
			\end{itemize}

\subsubsection{Gestion des animaux}L'utilisateur est amené sur la fenêtre de gestion des animaux, différentes options lui sont proposées:
			\vspace{3mm}\begin{itemize}
				\item nourrir tous les animaux d'un enclos spécifique
				\item soigner les animaux malades
				\item abattre un animal désigné et stocker sa viande
				\item récolter les ressources produites par un animal désigné
			\end{itemize}

\subsubsection{Gestion des cultures}L'utilisateur est amené sur la fenêtre de gestion des cultures, différentes options lui sont proposées:
			\vspace{3mm}\begin{itemize}
				\item arroser une parcelle spécifique
				\item faire la récolte d'une parcelle
			\end{itemize}
